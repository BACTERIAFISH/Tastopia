如果以一天三餐的頻率來看，一個人一年大概能進食一千次左右，在這有限的次數內，對我而言，每次的用餐都很珍貴。我喜歡到處品嚐各式各樣的食物，也喜歡與朋友分享我覺得不錯的餐廳，所以我製作了 Tastopia，讓使用者藉由分享食記的方式，來發掘更多不同的餐廳。
